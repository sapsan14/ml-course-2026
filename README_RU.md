## Что нужно

- **Python** 3.8+ (для локального запуска)
- **Аккаунт Google** (для Colab и сдачи в Moodle)
- **Cursor IDE** (удобно для правок) или любой редактор + терминал

### Пакеты Python (для локального Jupyter)

```bash
pip install pandas numpy jupyter pytz
```

## Workflow: Cursor и Google Colab

- **Разработка** в Cursor: правки в `ex1/EX01.ipynb`, запуск ячеек через расширение Jupyter.
- **Сдача** в Colab: загрузить `EX01.ipynb` и `dirty_dataset.csv` в Google Drive → открыть в Colab → Run all → отправить ссылку в Moodle.

Подробно по шагам: **[WORKFLOW_RU.md](WORKFLOW_RU.md)**.
