## Requirements

- **Python** 3.8+ (for local runs)
- **Google account** (for Colab and Moodle submission)
- **Cursor IDE** (recommended for editing) or any editor + terminal

### Python packages (for local Jupyter)

```bash
pip install pandas numpy jupyter pytz
```

---

## Workflow: Cursor vs Google Colab

- **Develop** in Cursor: edit `ex1/EX01.ipynb` (or `.py` with `# %%` cells), run cells locally with Jupyter extension.
- **Submit** in Colab: upload `EX01.ipynb` and `dirty_dataset.csv` to Google Drive → open in Colab → run all → share link in Moodle.

See **[WORKFLOW.md](WORKFLOW.md)** for step-by-step instructions (RU: **[WORKFLOW_RU.md](WORKFLOW_RU.md)**).
